# Nexsoft Portfolio Website

A responsive personal portfolio website built for the Nexsoft Solutions internship task.

## Live Demo

https://whimsical-cannoli-515ef2.netlify.app/

## About This Project

This portfolio website presents my profile as a frontend developer and Software Engineering student. It includes a responsive homepage, About section, Skills section, Projects section, Contact section, social links, resume download button, and footer.

## Features

- Responsive homepage design
- Fixed responsive navbar
- Dark neon cyberpunk UI
- About section with profile photo
- Skills section with grouped badges
- Projects section with screenshots
- Contact form with inline validation
- GitHub and LinkedIn links
- Resume download button
- Responsive footer
- Back to top button
- Smooth scrolling
- Active navbar highlighting
- Typewriter text animation

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery
- Google Fonts
